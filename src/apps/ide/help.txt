This is the help page for Pascal65.
Ctrl+N jumps to the next page and
Ctrl+P jumps to the previous page.
You can also use the cursor keys to
move around in this file.

Function             Escape  Ctrl Key
-----------------    ------  --------
Beginning of line    Esc+J    Ctrl+J
End of line          Esc+K    Ctrl+K
Page Up              Esc+Up   Ctrl+P
Page Down            Esc+Dn   Ctrl+N
Del. to start of ln  Esc+P
Del. to end of line  Esc+Q
Insert line          Esc+I
Scroll Up            Esc+V    Ctrl+V
Scroll Down          Esc+W    Ctrl+W
Scroll to file top   Esc+B    Ctrl+B
Scroll to file end   Esc+E    Ctrl+E
Mark Text                     Ctrl+Y
Paste Clipboard               Ctrl+U
Open File                     F1
Save File                     F2
Other File Functions      Back arrow
Run Pascal File               F3
Compile Pascal File           F5
View This Help File           F7
Exit                          Ctrl+X
